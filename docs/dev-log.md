# 开发日志

## 2026-07-19 · 0.1.6

- 查看区改为中性浅灰工作区与纯白文档表面，系统组件可用时继续优先还原 Office 原始版式。
- PDF 复用 WebView2 环境并验证真实导航结果，增加进程恢复和无 Runtime 时的明确错误提示。
- Office 兼容视图改为渐进渲染，修正 DOCX 图片尺寸，并降低大图解码和滚动开销。
- CSV/XLS/XLSX 增加流式读取、单元格预算和可取消的工作表切换，修复快速切换竞态。

## 2026-07-17 · 0.1.5

- 修正 DOCX 段落和表格顺序，取消伪分页并保留常用文字样式与内嵌图片。
- 增加全局缩放、最近记录删除和可持久化设置窗口。
- 增加系统高保真预览与统一缩放兼容视图的显示偏好。
- 移除检查更新入口、独立 updater 和在线更新发布资产。

## 2026-07-16 · 0.1.4

- 使用 Inno Setup 包装 Velopack 安装引导程序，形成经典 Windows 分步安装向导。
- 增加欢迎、安装说明、桌面快捷方式任务、准备安装和安装完成页面。
- 继续只交付一个 Setup EXE，Velopack 更新包作为在线更新后台资产。
- 生成 16 至 256px 多尺寸 ICO，统一主程序、更新程序、安装器和快捷方式图标。

## 2026-07-16

- 初始化 Git、.NET 8 分层解决方案与本地工具清单。
- 确定 WPF + WebView2 + Windows Preview Handler 技术路线，避免 Electron 常驻开销。
- 完成现代化无边框窗口、最近文件、拖放、快捷键和稳定加载状态。
- 完成 PDF、CSV、XLSX、Word、PowerPoint 与旧版 Excel 的查看路由。
- CSV 支持 UTF-8/UTF-16/GBK、分隔符检测、引号换行和 10 万行保护。
- XLSX 支持共享字符串、多工作表与 5 万行保护。
- 接入 Velopack 1.2.0，增加可配置更新源、安装器和差分包脚本。
- 完成 150% DPI 窗口截图检查，CSV 界面无裁切、重叠和空白渲染问题。
- 将 CSV 编码探测限制为前 64 KB，并将 XLSX 改为逐行流式读取。
- 完整解决方案零警告构建，格式识别、CSV、XLSX 共 3 项快速测试通过。
- 根据参考色板完成 0.1.1 视觉更新，并在 150% DPI 下重新截图验收。
- 将原应用内更新流程拆分为独立 `DocVista.Updater.exe`，由 Setup 与主程序一同安装。
- 配置 GitHub Releases 为生产更新源，并增加标签触发的自动发布流程。
- 复现安装版问题，新增 DOCX/PPTX/XLS 内置查看和 DOC/PPT 兼容文本回退。
- 为主程序与 updater 增加就绪握手，修复更新入口闪退。
- 修复窄窗口侧栏、长文件名和文档分页画布的错位问题。
