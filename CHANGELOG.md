# 项目更新说明

本项目遵循语义化版本，并按版本和发布日期记录面向用户的新增、优化与修复内容。

## 0.1.6 - 2026-07-19

### 修复

- PDF 查看器在窗口生命周期内复用同一个 WebView2 环境，等待真实导航结果，并在渲染进程退出时自动恢复。
- 修正 DOCX 图片尺寸单位，避免兼容视图把小图错误放大到整页。
- 快速切换文件或工作表时取消旧读取，避免旧结果覆盖新文档或并发读取同一个工作簿。

### 优化

- 文档工作区改为中性浅灰，PDF、表格和 Office 兼容页在没有源文件底色时统一使用纯白底色。
- Office 兼容视图按可视区域渐进创建内容并按显示尺寸解码图片，改善长文档首屏与滚动响应。
- CSV 改为小批探测后直接装载，XLSX 共享字符串改为流式读取，并为宽表增加单元格预览预算。

## 0.1.5 - 2026-07-17

### 修复

- DOCX 兼容视图按原文顺序显示段落和表格，不再把表格集中到文档末尾。
- 取消按固定段落数伪分页，保留标题、粗体、斜体、字号、对齐、缩进和内嵌图片。

### 新增

- PDF、表格、Office 兼容视图和系统预览增加统一缩放控件与快捷键。
- 最近打开记录增加单条删除功能，只删除记录，不删除原文件。
- 新增设置窗口，可调整默认缩放、缩放步长、缩放记忆、最近记录数量、表格行高、Office 显示模式和动画。

### 调整

- 移除“检查更新”和独立更新程序，后续版本通过新版 Setup 覆盖安装。
- GitHub Release 只发布经典 Windows Setup EXE。

## 0.1.4 - 2026-07-16

### 优化

- Setup 改用 Inno Setup 经典 Windows 分步安装向导，内部保留 Velopack 安装与更新能力。
- 新增欢迎、安装说明、桌面快捷方式选项、准备安装、安装进度和完成页面。
- 保持单一 Setup EXE、桌面/开始菜单快捷方式以及在线更新兼容性。
- 新增 DocVista 多尺寸应用图标，并应用到主程序、更新程序、Setup 和快捷方式。

## 0.1.3 - 2026-07-16

### 新增

- 侧栏新增“卸载 DocVista”，二次确认后调用 Setup 登记的系统卸载程序。
- Setup 安装后创建桌面和开始菜单快捷方式。

### 发布

- 本地交付目录只保留 `DocVista-win-Setup.exe`，更新包仅作为在线更新后台资产。

## 0.1.2 - 2026-07-16

### 修复

- 修复 DOC/DOCX/PPT/PPTX 在系统没有有效 Preview Handler 时无法打开的问题。
- 新增 DOCX 正文与表格、PPTX 幻灯片、旧版 DOC/PPT 兼容文本回退。
- 新增 XLS 内置工作表读取，不再依赖 Office 或 WPS 预览组件。
- 修复主程序启动 updater 后立即退出造成的闪退观感，改为就绪握手后切换。
- 修复长文件名侵占窗口按钮和窄窗口侧栏导致的页面错位。

### 发布

- 只生成 Setup 作为用户安装入口，不再生成 Portable。

## 0.1.1 - 2026-07-16

### 优化

- 按 Outer Space、Dark Pastel Blue、Bone、Wenge、Dark Lava 色板重做完整界面配色。
- 统一侧栏、窗口控制、工具栏、搜索、表格标题、交替行和选择状态的视觉层级。
- 安装程序进度色同步调整为 Dark Pastel Blue。
- 新增随 Setup 安装的独立 `DocVista.Updater.exe`，提供检查、下载、安装和重启界面。
- 主程序的“检查更新”改为启动独立更新程序，并退出以释放待更新文件。

## 0.1.0 - 2026-07-16

### 新增

- Windows 只读文档查看器基础版本。
- PDF、CSV、XLSX 内置查看，以及 DOC/DOCX/PPT/PPTX/XLS 系统预览。
- 文件拖放、最近文件、表格搜索、Excel 工作表切换和窗口状态恢复。
- Velopack 安装、完整更新包、差分更新包与应用内更新流程。
- 安装后加入系统“打开方式”，不抢占用户已有的默认文档应用。

### 已知限制

- DOC、PPT 和 XLS 需要系统注册 Microsoft Office 或兼容预览组件。
- XLSX 内置查看器首版聚焦单元格数据，不渲染图表、宏与复杂样式。
