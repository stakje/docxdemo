# Changelog

本项目遵循语义化版本。面向用户的新增、优化和修复记录在这里。

## 0.1.1 - 2026-07-16

### 优化

- 按 Outer Space、Dark Pastel Blue、Bone、Wenge、Dark Lava 色板重做完整界面配色。
- 统一侧栏、窗口控制、工具栏、搜索、表格标题、交替行和选择状态的视觉层级。
- 安装程序进度色同步调整为 Dark Pastel Blue。
- 新增随 Setup 安装的独立 `DocVista.Updater.exe`，提供检查、下载、安装和重启界面。
- 主程序的“检查更新”改为启动独立更新程序，并退出以释放待更新文件。

## 0.1.0 - 2026-07-16

### 新增

- Windows 只读文档查看器基础版本。
- PDF、CSV、XLSX 内置查看，以及 DOC/DOCX/PPT/PPTX/XLS 系统预览。
- 文件拖放、最近文件、表格搜索、Excel 工作表切换和窗口状态恢复。
- Velopack 安装、完整更新包、差分更新包与应用内更新流程。
- 安装后加入系统“打开方式”，不抢占用户已有的默认文档应用。

### 已知限制

- DOC、PPT 和 XLS 需要系统注册 Microsoft Office 或兼容预览组件。
- XLSX 内置查看器首版聚焦单元格数据，不渲染图表、宏与复杂样式。
